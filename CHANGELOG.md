# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-27

### Added
- Initial release of HTTP Problem Details middleware
- PSR-15 error handling middleware
- RFC 7807 Problem Details support
- Automatic exception to Problem Details conversion
- PSR-3 logging integration
- Customizable error response format
- Development vs production mode
- Stack trace inclusion (development mode)
- JSON problem details responses
- HTTP status code mapping
- Framework-agnostic implementation
- Comprehensive test suite
- Complete documentation with examples
